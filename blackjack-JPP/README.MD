# Blackjack Duel

A browser-based blackjack game built with HTML, CSS, and JavaScript.

## Goal

Start with 100 chips and reach 1000 chips to win the table.
If you lose all your chips, the game is over.

## Rules

- Player versus dealer
- Standard blackjack scoring
- 6-deck shoe
- Dealer stands on 17 or more
- Blackjack pays 3:2
- Push returns the bet
- No split, surrender, or double down in this version
- You can bet any whole number of chips you currently own

## Controls

- Enter your bet amount
- Click Deal to begin a round
- Click Hit to take another card
- Click Stand to end your turn
- Click New Round after the round resolves

## Run

Open `index.html` in a browser.
For best results, serve the folder with a simple static server.
